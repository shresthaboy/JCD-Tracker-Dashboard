# JCD Tracker Dashboard

Power BI dashboard for tracking Job Control Documents across Scaffolding, Insulation, and Painting disciplines at Qatar Shell GTL.

**Key Features:**
- Real-time JCD status (Submitted vs Approved)
- Tracks 350+ monthly documents
- Multi-level filtering by area, work centre, discipline
- Bottleneck visibility & trend analysis

**Tech:** Power BI | Power Query | DAX

**Impact:** Reduced processing time, improved visibility, better accuracy

---

To use: Download the `.pbix` file and open in Power BI Desktop.
